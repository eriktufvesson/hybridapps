Vad är en hybridapp?

  • En webbapplikation som visas i en inbyggd webbläsare (WebView)
  • Kan kommunicera med telefonens mjukvara/hårdvara via plugins
  • Utvecklas i HTML5/CSS3 och Javascript
  • En kodbas till både Android/iOS
  • Kompileras till ett installerarbart/distribuerbart paket

Vilka ramverk finns det?

  • Apache Cordova (tidigare PhoneGap)
  • Ionic Framework
  • Sencha Touch
  • Appcelerator
  • AppGyver
  • Onsen UI
  • React Native (Facebook)
  • NativeScript (Telerik)
  • Xamarin (C#)

Ionic Framework

Ionic Framework

  • Bygger på AngularJS
  • Byggt för att se ut och kännas som Native
  • Fokus på enkelhet och prestanda
  • Enkelt att kommunicera med Native plugins (ngCordova)

Prestanda

  • Hårdvaruaccelererade animationer
  • Tagit bort 300 ms fördröjningen vid klick
  • Minimal DOM-manipulation

Utseende

  • SASS
  • Anpassade gränssnitt för respektive plattform (iOS/Android)
  • Enkelt att customisera
  • Stort bibliotek av komponenter
  • Inbyggd ikon-font (Ionicons)

Complex Lists

  • AngularJS Directive
  • Buttons exposed by swiping
  • Reorder
  • Delete
